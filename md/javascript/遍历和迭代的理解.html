# 关于遍历和迭代的区别

在线性结构中按每个单元访问，这个过程叫做迭代(数组，栈，队列，链表) 在非线性结构中按每个单元访问。这个过程叫遍历(图，树)

js数组可以是线性结构（比如[1,2,3,4]），也可以是非线性结构（[1, true, {a: 1}]），所以js数组严格上不是真正的数组，真正的数组是存储在连续内存空间里
